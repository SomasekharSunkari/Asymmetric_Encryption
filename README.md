## Getting Started

### Prerequisites
* Java Development Kit (JDK) 8 or higher
* An IDE (IntelliJ IDEA, Eclipse) or a terminal with `javac`

### Usage
1. Clone the repository: `git clone https://github.com/[username]/Asymmetric_Encryption.git`
2. Open the project in your preferred Java environment.
3. Run the main class to generate RSA key pairs and encrypt/decrypt sample messages.
